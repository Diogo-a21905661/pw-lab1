HTML5

O HTML5 foi lançado em 2014, tendo o objetivo de simplificar vários aspectos do XHTML. Mantém grande parte das regras introduzidas pelo XHTML, tal como tudo estar em minúsculas e o correto encadeamento das etiquetas, mas também integrou um conjunto de funcionalidades que se revelaram úteis ao longo dos vários anos de sedimentação da web.

Sendo assim, o HTML5 acabou por ficar como uma versão mais simplificada do que o XHTML, mas mais exigentes que as versões anteriores de HTML.

CSS

O CSS surge para separar o contéudo da sua aparência. É um conjunto de regras, onde cada regra é formada por um "seletor", capaz de identificar elementos do documento, e por uma "declaração de estilo", que será o valor. O uso do CSS diminui o tempo de desenvolvimento, faz o código mais facil de escrever e dá melhor controlo sobre a colocação dos elementos nas páginas.


CSS - Seletores
Seletores Utilizou? Descrição
Universal circle O seletor universal aplica o estilo a todos os elementos da página.
Tipo circle O seletor de tipo usa o nome do marcador a estilizar, onde a regra será aplicada a todos os elementos que possuem esse marcador.
Classe circle O seletor de classe associa um nome a um estilo, onde o nome é antecedido de ".".
Id circle O atributo "id" permite identificar univocamente um elemento do HTML, onde o id é antecedido de "#".
Atributo circle Seletores que se aplicam a atributos de elementos, onde pode ser apenas referido o atributo.
Pseudo-Classe circle Seletor associado ao estado de um elemento.
Pseudo-Elemento circle Regras que se aplicam a partes de elementos.

CSS - Combinadores de Seletores
Combinadores de Seletores Utilizou? Descrição
Descendentes add Seletores descendentes são separados por espaço (div p). Todos os "p" que estão "dentro" de um "div".
Filhos add Seletores filhos através do uso de ">" (div > p). Todos os "p" que estão "logo dentro" de um "div".
Adjacentes add Seletores adjacentes através do uso de "~" (div ~ p). Todos os "p" que estão colocados depois de um "div".
Imediatamente Adjacentes add Seletores imediatamente adjacentes através do uso de "+" (div + p). Todos os "p" que estão colocados "imediatamente" depois de um "div".
Agrupar add Agrupar os seletores, separando-os por virgulas (div,p). Os seletores irão ficar com as condições.
